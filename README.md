## Getting Started

To get a local copy up and running follow these simple example steps.

To set up backend use

```
npm install
```

in the folder containing package.json

To set up frontend , cd to course_frontend and use

```
npm install
```

in the folder containing package.json

### Prerequisites

This project has been tested for node version >=18.

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/gullyboy007/imarticus_task
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
